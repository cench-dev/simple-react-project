# Academic Lock-in

Учебное React-приложение для подготовки к экзаменам.  
Позволяет открывать билеты, просматривать вопросы и ответы, а также добавлять билеты в избранное.

---

## Функционал

- Получение случайного билета  
- Открытие билета по номеру  
- Просмотр вопросов и ответов внутри билета  
- Показ / скрытие ответов по кнопке  
- Добавление и удаление билетов из избранного  
- Сохранение избранного в localStorage  
- Отображение списка избранных билетов  
- Навигация между страницами (React Router)  
- Простой UI (MVP проект)

---

## Технологии

- React  
- TypeScript  
- Vite  
- SCSS (Sass)  
- React Router DOM  
- LocalStorage API
